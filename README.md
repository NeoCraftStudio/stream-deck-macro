# Stream Deck Macro

Deck de macros DIY (estilo Elgato Stream Deck), baseado em Arduino Pro Micro
(ATmega32U4): matriz de botões 4x4 com camada secundária (2FX), 3 encoders
rotativos para controle de áudio, e LEDs WS2812B individuais por tecla.

Um app companheiro em Python (PySide6) recebe os eventos do firmware via
serial e executa as ações configuradas: atalhos de teclado, controle de cenas
do OBS, reprodução de som e mute real por aplicativo.

## Status

Em planejamento / estudo. Arquitetura de hardware e protocolo já definidas;
implementação ainda não iniciada.
